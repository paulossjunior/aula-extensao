# Materiais de Apoio

Esta página reúne os principais materiais de apoio do curso para consulta rápida. Aqui você encontra templates, exemplos preenchidos e guias que ajudam no planejamento e na documentação do produto ao longo das aulas.

---

## Templates

Use estes modelos como ponto de partida para organizar o trabalho do grupo:

- [Template de PRD](../modelos/prd-template.md) — Documento base para registrar problema, público, concorrência, métricas e oportunidade.
- [Template Geral de Aula](../modelos/template.md) — Estrutura base para documentar novas aulas ou encontros.
- [Template de Planejamento Scrum](../modelos/scrum-planejamento-template.md) — Modelo para backlog, sprints, épicos, histórias de usuário e tarefas.
- [Template de DSM](../modelos/dsm-template.md) — Modelo para mapear dependências entre módulos, funcionalidades e componentes.
